--------------------------------------------------Important--------------------------------------------------------------------------------------------------------------

Le projet ne fonctionne pas correctement avec la derniere version de gcc ( 6.3.0 ), mais de preference avec la version integré dans l'editeur DevC++ qui est gcc 4.9.2


Pour lancer le projet, il suffit de lancer le fichier Script.bat, Ce fichier lancera la compilation du code C et se chargera de créer les courbes sous GNUPLOT.

Ces courbes seront exporté en fichier PNG dans le repertoire de travail local du projet.

Bonne lecture !!
